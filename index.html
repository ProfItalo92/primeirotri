HTML - Estrutura da Web

HTML (HyperText Markup Language) é a linguagem de marcação usada para estruturar o conteúdo na web.

História: Criado por Tim Berners-Lee em 1989, o HTML foi a primeira linguagem usada para criar páginas web, permitindo a marcação de texto, imagens, links, listas e outros elementos essenciais.

Objetivo: O HTML tem como objetivo estruturar o conteúdo na web, organizando textos, imagens, links, formulários e outros elementos, sem lidar com a lógica de programação.

CSS - Estilizando a Web

CSS (Cascading Style Sheets) é utilizado para estilizar a aparência dos elementos HTML.

História: O CSS foi criado por Håkon Wium Lie e Bert Bos em 1994, com o objetivo de separar o conteúdo da web da sua aparência.

Objetivo: O CSS é responsável por estilizar os elementos HTML, controlando cores, fontes, tamanhos, espaçamentos e layout, separando o design da estrutura da página.

JavaScript - Tornando a Web Interativa

JavaScript é a linguagem de programação que adiciona interatividade e dinamismo às páginas web.

História: Criado por Brendan Eich em 1995, o JavaScript permite a interação do usuário com a página web, criando funcionalidades dinâmicas como animações, validações de formulários e mais.

Objetivo: JavaScript tem como objetivo adicionar comportamento interativo às páginas web, manipulando conteúdos, eventos de usuário e comunicação com servidores.
